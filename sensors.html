Sensors

Instructions: Open the browser development tools and navigate to the Sensors panel. Change the location or orientation of the device and see how the values change in the sample app below.

The following sample code is used to retrieve the orientation and timezone values from the browser. 


let orientationType = window.screen.orientation.type
let orientationAngle = window.screen.orientation.angle
let timezone = Intl.DateTimeFormat().resolvedOptions().timeZone;
