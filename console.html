Console

console.log()

Instructions: Open the browser development tools and navigate to the console tab. Click the buttons below and view the results in the console.

This example logs a simple message to the console, using the console.log() method. It's the most basic way to output information to the console. This example runs the following code: 


                counter = 0;
                function logToConsole(){
                    console.log(`The count is ${counter++}`);
                }

console.table()

The console.table() method logs an object or array as a table in the console output. Each property of the object is displayed as a value in the table. This example runs the following code: 


                pet = {ageInYears:2.1, name:"monster", type:"hermit crab"};
                console.table(pet);

console.assert()

The console.assert() method logs a message to the console if an assertion is false. If the assertion is true, nothing is logged to the console. This example runs the following code: 


                function assertExample(){
                    console.assert(1 === 2, "1 does not equal 2");
                }

console.dir()

The console.dir() method logs an object as a directory in the console output. Nested objects are displayed in a navigatable tree structure. This example runs the following code: 

            crab = {ageInYears:2.1, name:"monster", type:"hermit crab", origin: {country:"United States", state:"New Jersey", city:"Ocean City"}};
            console.dir(crab);

info, trace, warn, debug, error

The console object has functions for logging different messages for different needs.

time and timeEnd

The console.time() and console.timeEnd() functions can be used to measure the time it takes for a block of code to execute. This example runs the following code: 
            
                console.time('timer');
                setTimeout(() => {
                    console.timeEnd('timer');
                }, 1000);
The output in the console shows the name of the timer and the time (in milliseconds) it took to execute the code block.
